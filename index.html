What is Machine Learning?

Machine learning enables computers to learn from data without being explicitly programmed. The model finds patterns and makes predictions based on examples.

How Training Works

During training, the model adjusts its internal parameters to minimize errors. Each epoch processes the entire dataset, gradually improving accuracy.

Understanding Metrics

Loss: Measures prediction error (lower is better)
Accuracy: Percentage of correct predictions
Learning Rate: Speed of learning
